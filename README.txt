UPLOAD TO GITHUB
----------------
1. Open the steinbran.github.io repository.
2. Upload all files from this folder to the repository root.
3. Upload your four app icons into the images folder with these exact names:
   - sjRecorder.png
   - sjCropVideo.png
   - sjTracker.png
   - sjTrAnalyse.png
4. Commit the changes.

The pages still work before the icons are added. A simple letter placeholder is shown automatically.

IMPORTANT
---------
Keep the file names exactly as written because the links use those names.

The sjTracker card is marked Available and links to sjTracker.html.
